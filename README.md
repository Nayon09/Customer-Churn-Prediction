# Customer-Churn-Prediction

Built a customer churn prediction model using an artificial neural network (ANN) to identify customers likely to leave.
Processed and analyzed customer data with Pandas and NumPy for feature engineering and data cleaning.
Designed, trained, and evaluated the ANN in TensorFlow, optimizing performance through hyperparameter tuning.
Achieved strong predictive accuracy, providing actionable insights for customer retention strategies.
